Eclipse Keypop®

“The first open source APIs dedicated to contactless ticketing and access control applications”

Eclipse Keypop offers Java and C++ interfaces facilitating the standardization of modular and interoperable software solutions providing high-level access to card readers and card transactions.

These APIs are in compliance with the “Terminal” specifications developed by the Calypso Networks Association.
